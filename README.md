[我][me]的[开发环境是 Windows 系统][win-env]，
时常也会在 \*nix 系统上游荡，
本 repo 存了一些我常用的配置。

# 安装

* Unix 系统可以用 curl 命令一键安装：

```sh
    bash -c "$(curl -fsSL https://raw.githubusercontent.com/LKI/myconf/master/setup.sh)"
```
